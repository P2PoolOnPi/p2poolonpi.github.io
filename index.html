P2Pool on Raspberry Pi

Low Latency and Low Power Consumption P2Pool

I'm creating P2Pools that can run on a Raspberry Pi!

To get started you will need to do a little shopping:

  • Raspberry Pi B+
  • Raspberry Pi B+ Case - Others will do, but this is cheap and well made.
  • Power Supply
  • SDHC Micro SD Card
  • Ethernet Cable - Recommended.
  • WiFi Dongle - Not recommended, but if your only choice.

Install Raspbian with floating point support from the Raspberry Pi website.

You now have everything you need to start making your P2Pool. Check the main GitHub and choose your chosen P2Pool to build.
