P2Pool on Raspberry Pi

Low Latency and Low Power Consumption P2Pool

I'm creating P2Pools that can run on a Raspberry Pi!

To get started you will need to do a little shopping:

  • Raspberry Pi B+
  • Raspberry Pi B+ Case - Others will do, but this is cheap and well made.
  • Power Supply
  • SDHC Micro SD Card
  • Ethernet Cable - Recommended.
  • WiFi Dongle - Not recommended, but if your only choice.

Install Raspbian with floating point support from the Raspberry Pi website.

You now have everything you need to start making your P2Pool. Check the main GitHub and choose your chosen P2Pool to build.

FIRE NEWS

I do not hold Raspberry Pi or Jenkins responsible in any way for the events.

A few days ago I had everything set up running in Jenkins to connect to the Raspberry Pi and attempt to build each of the pools. It would then attempt to mine a share on the P2Pool before resulting in a pass or a failure.

Well it turns out I must have over stressed the Raspberry Pi with all this testing. This led to a small electrical fire in my testing lab. No serious damage was done, apart from a little breadboard and some cabling. However the Raspberry Pi is damaged beyond repair. I have contacted the retailer who has told me that fire is not covered in their replacement guide. I will contact Raspberry Pi directly, however I have little faith in getting this resolved.

This project is on hold until I can get a replacement Raspberry Pi and identify the fault that led to the electrical failure.
